== Description ==
Simply add invoice generator in your website by using [makeinvoice] shortcode. Generate and download invoice PDF quickly. Expand your site capability to create invoice for your users and send directly to their client/customer email.

== Installation ==
1. Install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Setup few settings in the menu "Make Invoice" then use shortcode [makeinvoice] to show the form interface in any pages you want.
